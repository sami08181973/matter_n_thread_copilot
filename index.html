Course Overview

This training package introduces the evolution of smart home IoT standards, Matter architecture, Thread networking, security, OTA updates, and a capstone project. It includes HTML documentation pages and runnable C/C++ examples suitable for learning and demonstration.

Author: Muhammad Samiullah | Founder & CTO, Embedded AI Design Labs Pvt Ltd | muhammadsami@embedailabs.com

  • Module 1: Introduction to IoT Connectivity & Matter
  • Module 2: Matter Architecture & Device Model
  • Module 3: Matter Commissioning & Interaction Model
  • Module 4: Matter Embedded Development
  • Module 5: Thread Protocol Fundamentals
  • Module 6: Thread Network Formation & Communication
  • Module 7: Security, OTA & Debugging
  • Module 8: Capstone Project

Modules

Module 1

Smart home evolution, IoT standards, Matter overview, and interoperability challenges.

Module 2

Matter architecture, node/endpoint/cluster/device-type model, and design guidance.

Module 3

Commissioning flow, QR code onboarding, secure sessions, and interaction model basics.

Module 4

Embedded development with ESP32/nRF, Matter SDK structure, and custom clusters.

Module 5

Thread networking concepts, roles, topology, and mesh principles.

Module 6

Thread network creation, joining, routing, addressing, and border router operations.

Module 7

Security, certificates, OTA updates, debugging, and troubleshooting.

Module 8

Capstone project for a Matter over Thread smart home solution.

Hardware & Tooling

  • ESP32 / nRF development boards for embedded demos.
  • Raspberry Pi for OpenThread Border Router (OTBR) and networking lab.
  • CHIP Tool for commissioning and testing Matter devices.
  • Wireshark for packet analysis and troubleshooting.
  • VS Code and ESP-IDF/Nordic SDK for firmware development.
