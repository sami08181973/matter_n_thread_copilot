Module 4: Matter Embedded Development

This module is focused on building Matter firmware for embedded platforms and customizing device capabilities.

Learning Goals

  • Understand the Matter SDK structure and firmware components.
  • Build sample applications for ESP32 or nRF targets.
  • Create custom clusters and extend an existing sample.

Recommended Setup

  • ESP-IDF for ESP32 boards
  • nRF Connect SDK for nRF targets
  • VS Code with the relevant toolchain installed

Hands-on Lab

Start from a Matter lighting sample and add a custom cluster that reports a simple device status or battery level.

Reference C++ Example

#include <iostream>
#include <string>

struct FirmwareComponent {
    std::string name;
    std::string role;
};

int main() {
    FirmwareComponent app{"Matter Application", "Handles device logic"};
    FirmwareComponent transport{"Network Stack", "Handles Matter transport"};
    std::cout << app.name << " - " << app.role << "\n";
    std::cout << transport.name << " - " << transport.role << "\n";
    return 0;
}
