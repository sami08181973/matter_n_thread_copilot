Module 6: Thread Network Formation & Communication

This module focuses on creating a Thread network, joining devices, routing, addressing, and border router operation.

Learning Goals

  • Configure and troubleshoot a Thread mesh network.
  • Understand unicast and multicast communication.
  • Explore border router operation and IPv6 addressing.

Hands-on Lab

Create a small Thread network using OpenThread and a Raspberry Pi border router. Add a sensor node and observe how packets flow through the mesh.

Reference C++ Example

#include <iostream>
#include <string>

struct ThreadPacket {
    std::string source;
    std::string destination;
    std::string type;
};

int main() {
    ThreadPacket packet{"SensorNode", "BorderRouter", "Unicast"};
    std::cout << packet.source << " -> " << packet.destination << " (" << packet.type << ")\n";
    return 0;
}
