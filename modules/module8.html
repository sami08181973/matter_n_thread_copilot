Module 8: Capstone Project – Matter over Thread Smart Home Solution

Build a production-ready smart home solution that combines a smart light, a smart plug, and an environmental sensor over Thread and Matter.

Project Scope

  • Implement a Matter light endpoint
  • Implement a Matter plug endpoint
  • Implement a sensor endpoint and telemetry reporting
  • Use a border router to connect the Thread network to a controller

Deliverables

  • System architecture diagram
  • Device model and cluster mapping
  • Firmware build and commissioning steps
  • Testing and validation report

Reference C++ Example

#include <iostream>
#include <string>

struct SmartHomeDevice {
    std::string role;
    std::string network;
};

int main() {
    SmartHomeDevice light{"Light", "Thread"};
    SmartHomeDevice plug{"Plug", "Thread"};
    SmartHomeDevice sensor{"Sensor", "Thread"};

    std::cout << light.role << " on " << light.network << "\n";
    std::cout << plug.role << " on " << plug.network << "\n";
    std::cout << sensor.role << " on " << sensor.network << "\n";
    return 0;
}
