Setup Guide

Follow these steps to prepare the hardware and software stack for the Matter and Thread training labs.

1. Development Environment

  • Install Visual Studio Code.
  • Install a C++ compiler such as g++ or MSYS2 on Windows.
  • Clone the Matter SDK from Connected Home IP.

2. ESP32 / nRF Boards

  • For ESP32, install ESP-IDF and configure the toolchain.
  • For nRF boards, install the nRF Connect SDK and VS Code extensions.
  • Purchase links: ESP32 and nRF52840 DK.

3. Raspberry Pi OpenThread Border Router

  • Use a Raspberry Pi 4 or 5 with a compatible power supply.
  • Install OpenThread Border Router (OTBR) and configure the network interface.
  • Purchase link: Raspberry Pi Products.

4. CHIP Tool and Matter Controller

  • Build or download CHIP Tool to commission and test Matter devices.
  • Use a mobile controller or desktop controller for onboarding and control.
  • Reference: Matter SDK / CHIP Tool.

5. Packet Analysis

  • Install Wireshark for network and commissioning traffic analysis.
  • Use it together with OpenThread sniffer tools when available.
